# Skills & Connectors Assignment Report

## Task 1 - Build Your First Real Skill

I created a personal Skill in Claude using the Skill Creator. I answered the clarifying questions, generated the SKILL.md file, and tested it in a new chat. The Skill triggered automatically from a normal request and produced the expected format.

## Task 2 - Connect One App, Read-Only

I connected Google Drive with read-only access. I asked the AI to summarize my most recent document and verified that the summary matched the original document. Read-only access was sufficient for this assignment, so I did not need write access.

## Task 3 - Wire Skill + Connector Together

I combined my Skill with the Google Drive connector. Using one plain-English request, the AI retrieved a document from Drive and formatted it according to my Skill. I checked the output against the original document to verify it was correct.

## Task 4 - Make It Portable or Hand It Off

I chose the portable option. I confirmed that the Skill worked outside the original conversation and produced the expected result without needing the instructions to be explained again.

## Task 5 - Audit a Skill Before Trusting It

I reviewed a Skill from the official skills directory and asked the AI to explain what it does and whether it performs any sensitive actions such as contacting external servers, handling credentials, or sharing data unexpectedly. Based on the explanation, I considered the Skill safe to enable for its intended purpose.

## Conclusion

This assignment taught me how to create reusable Skills, safely connect AI to real applications, combine Skills with Connectors, make a Skill portable, and evaluate a Skill from a security perspective.
